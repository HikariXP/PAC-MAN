# PAC-MAN
 这是一个吃豆人项目，但是在寻路上采用了最简单的A*算法（其实算不上），如果有需要可以直接download一个来做你想做的事情。

*注意：项目用的所有素材均开源。*



吃豆人的两个核心，网格移动和AI。

网格移动上是做的比较拉垮的，但是勉强是实现了。

AI上采用了类似于A星算法的思路，但是比较懒，只实现了一次的寻路。

也就是AI会基于一个目标提供器，然后只进行一次计算，算周围4格哪一格最近，而且非特殊情况无法扭头。（应该不算A*算法23333）



勉强是能用的。
